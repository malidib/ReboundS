# Binary Star Evolution Modules in REBOUNDx 
** Note: the current repo is temporary while the new modules are fully integrated into main REBOUNDx. **

Paper: https://arxiv.org/abs/2604.06386

How to install: 
- The following has been tested on python==3.13.2
- Install rebound==4.5.1
- Download this repo locally and install the modified REBOUNDx package using:  pip install -e ./ 
- Use REBOUND and REBOUNDx as normal.
- Examples are provided in binary_examples/

REBOUND: https://github.com/hannorein/rebound

REBOUNDx: https://github.com/dtamayo/reboundx
